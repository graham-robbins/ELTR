# Claude Code Instructions

## Commit Guidelines

**Never add "Co-Authored-By: Claude" or any similar co-author attribution to commits.**

When creating git commits, do not include:
- `Co-Authored-By: Claude ...`
- `Co-authored-by: Claude ...`
- Any variation attributing authorship to Claude, AI assistants, or Anthropic

Commits should only include human co-authors when applicable.
